Bro IPTV Player

最終更新日: 2026年4月23日

Bro IPTV Player は、iPhone、iPad、Mac、Apple TV 向けの M3U および M3U8 プレイリストプレーヤーです。 このアプリはアカウント登録を必要とせず、組み込みのチャンネル、 メディア、プレイリストは含まれていません。

お客様が提供する情報

  • プレイリスト URL、任意の EPG URL、任意の画像またはロゴ URL を追加したり、プレイリストファイルを読み込んだりできます。
  • これらの情報は、お客様が要求したコンテンツを読み込み、処理し、表示するためにのみ使用されます。
  • PIN 保護を有効にした場合、デバイスに保存されたプレイリストデータはローカルで暗号化されることがあります。

デバイスに保存される情報

  • 保存されたプレイリストと関連メタデータ。
  • アプリ設定および再生に関する設定。
  • アプリ機能に必要な場合の、キャッシュされたプレイリスト画像と番組ガイドデータ。

デバイス外に送信される情報

  • ユーザーが指定したプレイリスト、EPG、画像またはロゴの URL は、アプリに表示を求めたコンテンツを取得するためにのみネットワーク経由で要求されます。
  • Firebase Analytics は、アプリの利用状況とインストール状況を把握するために使用されます。
  • Firebase Crashlytics は、アプリの安定性向上に役立つクラッシュレポートと診断データを受け取るために使用されます。

データの利用方法

  • ユーザーが提供したプレイリストとストリームを読み込み、表示するため。
  • プレイリスト管理、再生、EPG、ロゴなどの機能を提供するため。
  • 集計レベルでアプリ利用を測定し、製品の挙動を把握するため。
  • クラッシュ、エラー、安定性の問題を診断するため。

広告とトラッキング

  • このアプリは第三者広告を表示しません。
  • このアプリは、アプリ間または Web サイト間のトラッキングのためにお客様のデータを使用しません。
  • このアプリは個人データを販売しません。

第三者サービス

このアプリは、Firebase Analytics および Firebase Crashlytics を含む Google Firebase サービスを使用します。

お客様の選択肢

  • いつでもアプリからプレイリストを削除できます。
  • プレイリスト、EPG、画像ソースを追加または読み込むかどうかを選択できます。
  • いつでもアプリの使用をやめ、デバイスからローカルのアプリデータを削除できます。

お問い合わせ

プライバシーまたはサポートに関する質問は、プロジェクトのリポジトリを通じて開発者までご連絡ください: GitHub Issues.

このポリシーは、Apple プラットフォームを通じて配布される Bro IPTV Player アプリに適用されます。アプリのデータ取扱いが変更された場合、このポリシーは更新されます。

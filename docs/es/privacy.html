Política de privacidad

Bro IPTV Player

Última actualización: 23 de abril de 2026

Bro IPTV Player es un reproductor de listas M3U y M3U8 para iPhone, iPad, Mac y Apple TV. La app no requiere registro de cuenta y no incluye canales, contenido multimedia ni listas integradas.

Información que proporcionas

  • Puedes agregar URL de listas, URL opcionales de EPG, URL opcionales de imágenes o logotipos, o importar archivos de listas.
  • Esta información se usa solo para cargar, procesar y mostrar el contenido que solicitas.
  • Si activas la protección con PIN, los datos de las listas almacenados en el dispositivo pueden cifrarse localmente.

Información almacenada en tu dispositivo

  • Listas guardadas y metadatos relacionados.
  • Ajustes de la app y preferencias relacionadas con la reproducción.
  • Imágenes de listas almacenadas en caché y datos de la guía de programas cuando son necesarios para las funciones de la app.

Información enviada fuera del dispositivo

  • Las URL de listas, EPG e imágenes o logotipos proporcionadas por el usuario se solicitan por la red solo para obtener el contenido que pides a la app que muestre.
  • Firebase Analytics se utiliza para comprender el uso y las instalaciones de la app.
  • Firebase Crashlytics se utiliza para recibir informes de fallos y datos de diagnóstico que ayudan a mejorar la estabilidad de la app.

Cómo se usan los datos

  • Para cargar y mostrar listas y transmisiones proporcionadas por el usuario.
  • Para ofrecer funciones como gestión de listas, reproducción, EPG y logotipos.
  • Para medir el uso de la app a nivel agregado y comprender el comportamiento del producto.
  • Para diagnosticar fallos, errores y problemas de estabilidad.

Publicidad y seguimiento

  • La app no muestra anuncios de terceros.
  • La app no utiliza tus datos para rastreo entre apps o sitios web.
  • La app no vende datos personales.

Servicios de terceros

La app utiliza servicios de Google Firebase, incluidos Firebase Analytics y Firebase Crashlytics.

Tus opciones

  • Puedes eliminar listas de la app en cualquier momento.
  • Puedes decidir si deseas agregar o importar una lista, un EPG o una fuente de imágenes.
  • Puedes dejar de usar la app en cualquier momento y eliminar los datos locales de la app de tu dispositivo.

Contacto

Para preguntas sobre privacidad o soporte, contacta con el desarrollador a través del repositorio del proyecto: GitHub Issues.

Esta política se aplica a la app Bro IPTV Player distribuida a través de las plataformas de Apple. Si cambian las prácticas de datos de la app, esta política se actualizará.

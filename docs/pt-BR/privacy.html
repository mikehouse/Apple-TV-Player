Política de Privacidade

Bro IPTV Player

Última atualização: 23 de abril de 2026

Bro IPTV Player é um reprodutor de playlists M3U e M3U8 para iPhone, iPad, Mac e Apple TV. O app não exige cadastro de conta e não inclui canais, mídia ou playlists integrados.

Informações que você fornece

  • Você pode adicionar URLs de playlists, URLs opcionais de EPG, URLs opcionais de imagens ou logotipos, ou importar arquivos de playlist.
  • Essas informações são usadas apenas para carregar, processar e exibir o conteúdo solicitado por você.
  • Se você ativar a proteção por PIN, os dados de playlist armazenados no dispositivo poderão ser criptografados localmente.

Informações armazenadas no seu dispositivo

  • Playlists salvas e metadados relacionados.
  • Configurações do app e preferências relacionadas à reprodução.
  • Imagens de playlist em cache e dados do guia de programação quando necessário para os recursos do app.

Informações enviadas para fora do dispositivo

  • As URLs de playlists, EPG e imagens ou logotipos fornecidas pelo usuário são solicitadas pela rede apenas para buscar o conteúdo que você pede para o app exibir.
  • Firebase Analytics é usado para entender o uso do app e as instalações.
  • Firebase Crashlytics é usado para receber relatórios de falhas e dados de diagnóstico que ajudam a melhorar a estabilidade do app.

Como os dados são usados

  • Para carregar e exibir playlists e streams fornecidos pelo usuário.
  • Para oferecer recursos como gerenciamento de playlists, reprodução, EPG e logotipos.
  • Para medir o uso do app em nível agregado e entender o comportamento do produto.
  • Para diagnosticar falhas, erros e problemas de estabilidade.

Publicidade e rastreamento

  • O app não exibe anúncios de terceiros.
  • O app não usa seus dados para rastreamento entre apps ou sites.
  • O app não vende dados pessoais.

Serviços de terceiros

O app usa serviços do Google Firebase, incluindo Firebase Analytics e Firebase Crashlytics.

Suas escolhas

  • Você pode remover playlists do app a qualquer momento.
  • Você pode decidir se deseja adicionar ou importar uma playlist, um EPG ou uma fonte de imagens.
  • Você pode parar de usar o app a qualquer momento e remover os dados locais do app do seu dispositivo.

Contato

Para questões de privacidade ou suporte, entre em contato com o desenvolvedor pelo repositório do projeto: GitHub Issues.

Esta política se aplica ao app Bro IPTV Player distribuído pelas plataformas Apple. Se as práticas de dados do app mudarem, esta política será atualizada.

隐私政策

Bro IPTV Player

最后更新：2026年4月23日

Bro IPTV Player 是一款适用于 iPhone、iPad、Mac 和 Apple TV 的 M3U 和 M3U8 播放列表播放器。 该应用无需注册账户，也不包含内置频道、 媒体内容或播放列表。

您提供的信息

  • 您可以添加播放列表 URL、可选的 EPG URL、可选的图片或徽标 URL，或导入播放列表文件。
  • 这些信息仅用于加载、处理并显示您请求的内容。
  • 如果您启用 PIN 保护，存储在设备上的播放列表数据可能会在本地加密。

存储在您设备上的信息

  • 已保存的播放列表及相关元数据。
  • 应用设置和与播放相关的偏好。
  • 应用功能所需的缓存播放列表图片和节目指南数据。

发送到设备外部的信息

  • 用户提供的播放列表、EPG 以及图片或徽标 URL 仅会通过网络请求，用于获取您要求应用显示的内容。
  • Firebase Analytics 用于了解应用使用情况和安装情况。
  • Firebase Crashlytics 用于接收崩溃报告和诊断数据，以帮助提升应用稳定性。

数据如何使用

  • 用于加载并显示用户提供的播放列表和流。
  • 用于提供播放列表管理、播放、EPG 和徽标等功能。
  • 用于在汇总层面衡量应用使用情况并了解产品行为。
  • 用于诊断崩溃、错误和稳定性问题。

广告与跟踪

  • 该应用不会显示第三方广告。
  • 该应用不会将您的数据用于跨应用或跨网站跟踪。
  • 该应用不会出售个人数据。

第三方服务

该应用使用 Google Firebase 服务，包括 Firebase Analytics 和 Firebase Crashlytics。

您的选择

  • 您可以随时从应用中删除播放列表。
  • 您可以选择是否添加或导入播放列表、EPG 或图片来源。
  • 您可以随时停止使用该应用，并从设备中删除本地应用数据。

联系方式

如有隐私或支持相关问题，请通过项目仓库联系开发者： GitHub Issues.

本政策适用于通过 Apple 平台分发的 Bro IPTV Player 应用。如果该应用的数据处理方式发生变化，本政策将会更新。

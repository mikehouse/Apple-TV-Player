Политика конфиденциальности

Bro IPTV Player

Последнее обновление: 23 апреля 2026 г.

Bro IPTV Player — это плеер плейлистов M3U и M3U8 для iPhone, iPad, Mac и Apple TV. Приложение не требует регистрации учетной записи и не содержит встроенных каналов, медиаконтента или плейлистов.

Информация, которую вы предоставляете

  • Вы можете добавить URL плейлистов, необязательные URL EPG, необязательные URL изображений или логотипов либо импортировать файлы плейлистов.
  • Эта информация используется только для загрузки, обработки и отображения контента, который вы запросили.
  • Если вы включите защиту PIN-кодом, данные плейлистов, сохраненные на устройстве, могут быть зашифрованы локально.

Информация, сохраняемая на вашем устройстве

  • Сохраненные плейлисты и связанные с ними метаданные.
  • Настройки приложения и параметры, связанные с воспроизведением.
  • Кэшированные изображения плейлистов и данные программы передач, когда это необходимо для функций приложения.

Информация, передаваемая за пределы устройства

  • Предоставленные пользователем URL плейлистов, EPG и изображений или логотипов запрашиваются по сети только для получения контента, который вы попросили приложение отобразить.
  • Firebase Analytics используется для понимания того, как используется приложение, и для учета установок.
  • Firebase Crashlytics используется для получения отчетов о сбоях и диагностических данных, которые помогают улучшать стабильность приложения.

Как используются данные

  • Для загрузки и отображения предоставленных пользователем плейлистов и потоков.
  • Для предоставления функций, таких как управление плейлистами, воспроизведение, EPG и логотипы.
  • Для измерения использования приложения на агрегированном уровне и понимания поведения продукта.
  • Для диагностики сбоев, ошибок и проблем со стабильностью.

Реклама и отслеживание

  • Приложение не показывает стороннюю рекламу.
  • Приложение не использует ваши данные для отслеживания между приложениями или веб-сайтами.
  • Приложение не продает персональные данные.

Сторонние сервисы

Приложение использует сервисы Google Firebase, включая Firebase Analytics и Firebase Crashlytics.

Ваши возможности

  • Вы можете удалить плейлисты из приложения в любое время.
  • Вы сами решаете, добавлять или импортировать ли плейлист, EPG или источник изображений.
  • Вы можете прекратить использование приложения в любое время и удалить локальные данные приложения со своего устройства.

Контакты

По вопросам конфиденциальности или поддержки свяжитесь с разработчиком через репозиторий проекта: Issues на GitHub.

Эта политика распространяется на приложение Bro IPTV Player, распространяемое через платформы Apple. Если порядок обработки данных в приложении изменится, эта политика будет обновлена.

Bro IPTV Player

อัปเดตล่าสุด: 23 เมษายน 2026

Bro IPTV Player เป็นเครื่องเล่นเพลย์ลิสต์ M3U และ M3U8 สำหรับ iPhone, iPad, Mac และ Apple TV แอปนี้ไม่ต้องลงทะเบียนบัญชี และไม่มีช่อง, สื่อ หรือเพลย์ลิสต์ที่ติดตั้งมาในตัว

ข้อมูลที่คุณให้

  • คุณสามารถเพิ่ม URL เพลย์ลิสต์, URL EPG แบบไม่บังคับ, URL รูปภาพหรือโลโก้แบบไม่บังคับ หรือจะนำเข้าไฟล์เพลย์ลิสต์ก็ได้
  • ข้อมูลนี้ใช้เพื่อโหลด ประมวลผล และแสดงเนื้อหาที่คุณร้องขอเท่านั้น
  • หากคุณเปิดใช้การป้องกันด้วย PIN ข้อมูลเพลย์ลิสต์ที่เก็บไว้ในอุปกรณ์อาจถูกเข้ารหัสภายในเครื่อง

ข้อมูลที่จัดเก็บไว้ในอุปกรณ์ของคุณ

  • เพลย์ลิสต์ที่บันทึกไว้และข้อมูลเมตาที่เกี่ยวข้อง
  • การตั้งค่าแอปและค่ากำหนดที่เกี่ยวข้องกับการเล่น
  • รูปภาพเพลย์ลิสต์ที่แคชไว้และข้อมูลผังรายการเมื่อจำเป็นต่อฟีเจอร์ของแอป

ข้อมูลที่ส่งออกนอกอุปกรณ์

  • URL เพลย์ลิสต์, EPG และรูปภาพหรือโลโก้ที่ผู้ใช้ระบุ จะถูกเรียกผ่านเครือข่ายเฉพาะเพื่อดึงเนื้อหาที่คุณสั่งให้แอปแสดงเท่านั้น
  • Firebase Analytics ใช้เพื่อทำความเข้าใจการใช้งานแอปและการติดตั้ง
  • Firebase Crashlytics ใช้เพื่อรับรายงานการขัดข้องและข้อมูลวินิจฉัยที่ช่วยปรับปรุงเสถียรภาพของแอป

การใช้ข้อมูล

  • เพื่อโหลดและแสดงเพลย์ลิสต์และสตรีมที่ผู้ใช้ให้มา
  • เพื่อให้บริการฟีเจอร์ต่างๆ เช่น การจัดการเพลย์ลิสต์ การเล่น EPG และโลโก้
  • เพื่อวัดการใช้งานแอปในภาพรวมและทำความเข้าใจพฤติกรรมของผลิตภัณฑ์
  • เพื่อวินิจฉัยการขัดข้อง ข้อผิดพลาด และปัญหาด้านเสถียรภาพ

โฆษณาและการติดตาม

  • แอปนี้ไม่แสดงโฆษณาจากบุคคลที่สาม
  • แอปนี้ไม่ใช้ข้อมูลของคุณเพื่อติดตามข้ามแอปหรือข้ามเว็บไซต์
  • แอปนี้ไม่ขายข้อมูลส่วนบุคคล

บริการของบุคคลที่สาม

แอปนี้ใช้บริการ Google Firebase รวมถึง Firebase Analytics และ Firebase Crashlytics

ตัวเลือกของคุณ

  • คุณสามารถลบเพลย์ลิสต์ออกจากแอปได้ทุกเมื่อ
  • คุณสามารถเลือกได้ว่าจะเพิ่มหรือนำเข้าเพลย์ลิสต์, EPG หรือแหล่งรูปภาพหรือไม่
  • คุณสามารถหยุดใช้แอปได้ทุกเมื่อ และลบข้อมูลแอปที่จัดเก็บไว้ภายในอุปกรณ์ของคุณ

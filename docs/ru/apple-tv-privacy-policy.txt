Политика конфиденциальности

Последнее обновление: 23 апреля 2026 г.

Bro IPTV Player — это плеер плейлистов M3U и M3U8 для iPhone, iPad, Mac и Apple TV. Приложение не требует регистрации учетной записи и не содержит встроенных каналов, медиаконтента или плейлистов.

Информация, которую вы можете предоставить:
- URL плейлистов
- Необязательные URL EPG
- Необязательные URL изображений или логотипов
- Импортированные файлы плейлистов

Эта информация используется только для загрузки, обработки и отображения контента, который вы запросили.

Информация, сохраняемая на вашем устройстве, может включать:
- Сохраненные плейлисты и связанные с ними метаданные
- Настройки приложения и параметры воспроизведения
- Кэшированные изображения плейлистов и данные программы передач

Информация, передаваемая за пределы устройства:
- Предоставленные пользователем URL плейлистов, EPG и изображений или логотипов запрашиваются только для получения контента, который вы выбираете для загрузки
- Firebase Analytics используется для понимания того, как используется приложение, и для учета установок
- Firebase Crashlytics используется для получения отчетов о сбоях и диагностических данных с целью повышения стабильности приложения

Как используются данные:
- Для загрузки и отображения предоставленных пользователем плейлистов и потоков
- Для предоставления функций, таких как управление плейлистами, воспроизведение, EPG и логотипы
- Для измерения использования приложения на агрегированном уровне и понимания поведения продукта
- Для диагностики сбоев, ошибок и проблем со стабильностью

Реклама и отслеживание:
- Приложение не показывает стороннюю рекламу
- Приложение не использует ваши данные для отслеживания между приложениями или веб-сайтами
- Приложение не продает персональные данные

Сторонние сервисы:
- Google Firebase Analytics
- Google Firebase Crashlytics

Контакты:

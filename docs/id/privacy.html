Kebijakan Privasi

Bro IPTV Player

Terakhir diperbarui: 23 April 2026

Bro IPTV Player adalah pemutar playlist M3U dan M3U8 untuk iPhone, iPad, Mac, dan Apple TV. Aplikasi ini tidak memerlukan pendaftaran akun dan tidak menyertakan saluran, media, atau playlist bawaan.

Informasi yang Anda berikan

  • Anda dapat menambahkan URL playlist, URL EPG opsional, URL gambar atau logo opsional, atau mengimpor file playlist.
  • Informasi ini hanya digunakan untuk memuat, memproses, dan menampilkan konten yang Anda minta.
  • Jika Anda mengaktifkan perlindungan PIN, data playlist yang disimpan di perangkat dapat dienkripsi secara lokal.

Informasi yang disimpan di perangkat Anda

  • Playlist yang disimpan dan metadata terkait.
  • Pengaturan aplikasi dan preferensi terkait pemutaran.
  • Gambar playlist yang di-cache dan data panduan program saat diperlukan untuk fitur aplikasi.

Informasi yang dikirim ke luar perangkat

  • URL playlist, EPG, dan gambar atau logo yang diberikan pengguna diminta melalui jaringan hanya untuk mengambil konten yang Anda minta agar ditampilkan oleh aplikasi.
  • Firebase Analytics digunakan untuk memahami penggunaan aplikasi dan instalasi.
  • Firebase Crashlytics digunakan untuk menerima laporan crash dan data diagnostik yang membantu meningkatkan stabilitas aplikasi.

Bagaimana data digunakan

  • Untuk memuat dan menampilkan playlist serta stream yang diberikan pengguna.
  • Untuk menyediakan fitur seperti pengelolaan playlist, pemutaran, EPG, dan logo.
  • Untuk mengukur penggunaan aplikasi pada tingkat agregat dan memahami perilaku produk.
  • Untuk mendiagnosis crash, error, dan masalah stabilitas.

Iklan dan pelacakan

  • Aplikasi ini tidak menampilkan iklan pihak ketiga.
  • Aplikasi ini tidak menggunakan data Anda untuk pelacakan lintas aplikasi atau lintas situs web.
  • Aplikasi ini tidak menjual data pribadi.

Layanan pihak ketiga

Aplikasi ini menggunakan layanan Google Firebase, termasuk Firebase Analytics dan Firebase Crashlytics.

Pilihan Anda

  • Anda dapat menghapus playlist dari aplikasi kapan saja.
  • Anda dapat memilih apakah akan menambahkan atau mengimpor playlist, EPG, atau sumber gambar.
  • Anda dapat berhenti menggunakan aplikasi kapan saja dan menghapus data aplikasi lokal dari perangkat Anda.

Kontak

Untuk pertanyaan privasi atau dukungan, hubungi pengembang melalui repositori proyek: GitHub Issues.

Kebijakan ini berlaku untuk aplikasi Bro IPTV Player yang didistribusikan melalui platform Apple. Jika praktik data aplikasi berubah, kebijakan ini akan diperbarui.
